## 주요 기능

- 플레이어는 컴퓨터와 총 10회 가위 바위 보 게임을 할 수 있습니다.
- 각 라운드마다 플레이어와 컴퓨터의 선택에 따라 승, 패, 무승부를 알 수 있습니다.
- 10회 게임이 종료되면 누가 이겼는지 최종 결과를 알려줍니다.
